## 📋 PROMPT À COPIER-COLLER DANS KIMI

```
Je suis étudiant en informatique et je prépare ma soutenance de mémoire de fin d'études. 
J'ai besoin que tu m'aides à créer une présentation PowerPoint professionnelle et percutante.

## CONTEXTE DU PROJET

**Titre du projet :** TIRAHOU - Système Intégré de Gestion Universitaire Hybride

**Type de projet :** Plateforme web complète de gestion universitaire

**Stack technique :**
- Backend : Django 5.2 + Django REST Framework + PostgreSQL
- Frontend : React 19 + TypeScript + Vite + TailwindCSS
- DevOps : Docker + GitHub Actions + Celery + Redis

**Chiffres clés du projet :**
- 19 modules backend complets
- 87 modèles de base de données
- 150+ endpoints API RESTful
- 80+ composants React
- ~50,000 lignes de code
- 13 rôles utilisateurs différents
- Conformité au cahier des charges : 95.8%
- Documentation du code : 95%

**Fonctionnalités principales :**
1. Gestion académique complète (système LMD)
2. Inscriptions et admissions en ligne
3. Évaluations et notes (avec workflow complet)
4. Classes virtuelles hybrides (5 providers : BBB, Jitsi, Zoom, Meet, Teams)
5. LMS intégré (Learning Management System)
6. Gestion financière (paiements, bourses, échéanciers)
7. GED (Gestion Électronique des Documents) avec QR codes
8. Analytics prédictifs (détection risque d'échec)
9. Communication et notifications (multi-canal)
10. Bibliothèque numérique

**Points forts techniques :**
- Mode hybride natif (présentiel + distanciel synchrone/asynchrone)
- Analytics prédictifs avec ML-ready
- Système LMD complet avec compensation et capitalisation
- Documentation exhaustive (95% vs 70% standard industrie)
- Architecture modulaire avec Service Layer Pattern
- Validation multi-niveaux (Frontend + Backend + BDD)
- RBAC granulaire (13 rôles avec permissions détaillées)

**Innovations :**
- Prédiction du risque d'échec des étudiants (score 0-100)
- Statistiques avancées (quartiles, corrélations, outliers)
- Détection précoce étudiants en difficulté
- Workflow de notes complet (saisie → validation → publication)
- Génération automatique de documents avec QR codes

## CE QUE JE VEUX

Créer une présentation PowerPoint de **25-30 diapositives** pour une soutenance de **20-25 minutes**.

## STYLE ET DESIGN DEMANDÉS

**Palette de couleurs :**
- Couleur principale : Bleu (#3b82f6 - professionnel, tech)
- Couleur secondaire : Émeraude (#10b981 - réussite, innovation)
- Accent : Orange (#f97316 - énergie)
- Neutre : Gris foncé (#1f2937) et blanc

**Style visuel :**
- Design moderne et épuré
- Beaucoup d'icônes et de visuels
- Peu de texte par diapo (bullet points concis)
- Graphiques et diagrammes pour les chiffres
- Schémas d'architecture clairs
- Captures d'écran mockups si nécessaire

**Typographie :**
- Titres : Grande taille, gras, couleur principale
- Corps : Taille lisible, police sans-serif moderne
- Code (si nécessaire) : Police monospace

**Mise en page :**
- Beaucoup d'espace blanc
- Maximum 5-6 bullet points par diapo
- Visuels à droite ou en fond
- Numérotation des diapositives

## FORMAT DE SORTIE

Fournis-moi le contenu de CHAQUE diapositive sous ce format :

```

**Contenu textuel :**
- Point 1
- Point 2
- Point 3

**Éléments visuels suggérés :**
- Icône / Image / Graphique à inclure
- Couleur dominante de la diapo

```

## INSTRUCTIONS IMPORTANTES

1. **Concision :** Maximum 5-6 points par diapo, phrases courtes
2. **Visuels :** Suggère des icônes/graphiques pertinents
3. **Storytelling :** Crée une narration fluide et logique
4. **Données :** Utilise les chiffres clés fournis
5. **Équilibre :** Technique + Impact + Innovation
6. **Timing :** Notes du présentateur pour respecter 20-25 min
7. **Professionnalisme :** Vocabulaire académique mais accessible

## COMMENCE MAINTENANT

Génère-moi le contenu complet de toutes les diapositives en suivant exactement la structure demandée.
```
